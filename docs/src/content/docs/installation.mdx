## Getting Started

<Steps>
<ol class="!mt-10">
  <li>
    ### Installation

    <Tabs>
      <TabItem label="npm">
        ```ansi copy
        npm install astro-cloudinary@1.0.0-beta.14
        ```
      </TabItem>
      <TabItem label="pnpm">
        ```ansi copy
        pnpm install astro-cloudinary@1.0.0-beta.14
        ```
      </TabItem>
      <TabItem label="yarn">
        ```ansi copy
        yarn add astro-cloudinary@1.0.0-beta.14
        ```
      </TabItem>
    </Tabs>
  </li>
  <li>
    ### Configuration

    Add the following variable to your `.env` file.

    ```ansi copy
    PUBLIC_CLOUDINARY_CLOUD_NAME="<Your Cloud Name>"
    ```

    If using Signed Upload with the CldUploadWidget, additionally add:

    ```ansi copy
    PUBLIC_CLOUDINARY_API_KEY="<Your API Key>"
    CLOUDINARY_API_SECRET="<Your API Secret>"
    ```

    :::tip
    Don't have a Cloudinary account? <a href="https://cloudinary.com/users/register_free?utm_campaign=devx_astrocloudinary&utm_medium=referral&utm_source=astrocloudinary">Sign up for free</a> on cloudinary.com!
    :::
  </li>
</ol>
</Steps>
